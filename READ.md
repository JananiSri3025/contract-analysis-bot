# Contract Analysis & Risk Assessment Bot

This project is a simple AI-powered contract analysis tool designed for small and medium businesses.

## Features
- Upload contract text files (.txt)
- Detect important legal clauses
- Identify contract type
- Assess overall contract risk
- Provide plain English summaries
- Maintain audit logs

## Supported Contract Types
- Lease Agreement
- Vendor Agreement
- Service Agreement
- Employment Agreement

## Technology Used
- Python
- Streamlit
- Rule-based NLP

## How to Run Locally
1. Install Python 3.10+
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   streamlit run app.py

## Hackathon Problem Statement
Build a GenAI-powered legal assistant to help SMEs understand contracts and legal risks.
